## 合规摘要（电竞版）

不代练、不租号买卖账号、不外挂、不涉赌；**不保证**段位/胜率；区分 **事实（官方/赛果）** 与 **观点（梯度/评价）**；避免绝对化表述；必要时输出 **风险提示块**。
